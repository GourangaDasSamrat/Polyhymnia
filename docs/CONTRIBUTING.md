# Contributing

Thank you for considering contributing! Please follow these guidelines:

- Use the issue templates to report bugs, request features, or ask questions.
- For code changes, open a pull request using the provided `PULL_REQUEST_TEMPLATE.md`.
- Ensure tests pass and add tests for new behavior when possible.
- Follow the repository's coding style and run linters/formatters before submitting.

We use issue templates located in `.github/ISSUE_TEMPLATE/` to help triage and prioritize requests.
